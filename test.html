DOT渲染器测试页面

这个页面包含几个DOT语言示例，可以用来测试DOT渲染浏览器插件。

纯文本形式的```dot代码块测试

下面是一个直接使用Markdown格式的DOT代码块：

```dot digraph Simple { A -> B -> C; A -> C [label="直接路径", color=red]; } ```

简单有向图示例

digraph G {
  A -> B;
  B -> C;
  C -> D;
  D -> A;
}

带样式的图示例

digraph G {
  // 节点样式
  node [shape=box, style=filled, color=lightblue];
  
  // 边样式
  edge [color=grey, arrowhead=open];
  
  // 节点
  开始 [shape=oval];
  处理 [shape=box];
  决策 [shape=diamond];
  结束 [shape=oval];
  
  // 边
  开始 -> 处理;
  处理 -> 决策;
  决策 -> 处理 [label="否"];
  决策 -> 结束 [label="是"];
}

复杂图示例

digraph G {
  rankdir=LR;
  
  subgraph cluster_0 {
    style=filled;
    color=lightgrey;
    node [style=filled,color=white];
    a0 -> a1 -> a2 -> a3;
    label = "流程 A";
  }
  
  subgraph cluster_1 {
    node [style=filled];
    b0 -> b1 -> b2 -> b3;
    label = "流程 B";
    color=blue;
  }
  
  start -> a0;
  start -> b0;
  a1 -> b3;
  b2 -> a3;
  a3 -> end;
  b3 -> end;
  
  start [shape=Mdiamond];
  end [shape=Msquare];
}
